Example 2: JavaScript Touch Detection

How it works:

This approach uses JavaScript to detect touch support:

  • Touch detected: Assumes no hover support
  • No touch detected: Assumes hover support

Problems with this approach:

  • Hybrid devices (laptops with touchscreens) support both
  • Forces devices into binary categories
  • User might switch between input methods
  • False positives: Modern browsers (Chrome, Safari) include touch APIs even on non-touch devices like Macs
  • Browser compatibility features can trigger false touch detection

JavaScript Code:

const hasTouch = "ontouchstart" in window || navigator.maxTouchPoints > 0;

if (hasTouch) {
  document.body.classList.add("no-hover");
} else {
  document.body.classList.add("hover");
}
